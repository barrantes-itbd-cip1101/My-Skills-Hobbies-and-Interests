CIP 1101 - ITBD


Displaying Italicized Text.

1.) <i></i>
Physical tag will display element in italic format


2.) <em></em>
Logical tag to emphasize text in italic format.


3.) <cite></cite>
Logical tag for citation.


4.) <var></var>
The logical element indicates a variable, as might be used in computer code.


5.) <dfn></dfn>
This logical element identifies a portion of text defining instance of a term.


6.) <address></address>
Defines
contact information
for the author/owner of a document.


Displaying Boldface Text.

1.) <b></b>
The bold element is a physical element that allows you to render boldface text.

2.) <strong></strong>
This logical element indicates a heavier or stronger emphasis than does the number 1, but there is usually no difference in how the two look in the browser.


Displaying Superscript Text.

1.) <sup></sup>
This element creates a superscript.


Displaying Subscript Text.

2.) <sub></sub>
This element forces text to display as a subscript.


Displaying Monospaced Text.

1.) <code></code>
This logical element not only displays in a courier, fixed width-font font, but also indicates that the text is a portion of computer code.

2.) <kbd></kbd>
Standing for keyboard, this logical element identifies its contents as keyboard input.

3.) <samp></samp>
This is another logical element. It describe its content as sample output, most often rendering text in monospaced font.


Displaying Strikethroughs.

1.) <s></s>
This physical element will display text as “strikethrough”.

2.) <del></del>
The deleted element is a logical element that indicates that the text has been deleted but left in the page.


Displaying Underlined Text.

1.) <u></u>
The underline element will render text with a line underneath.

2.) <ins></ins>
The inserted text element is a logical element that indicates text that has been inserted since the document was written.
